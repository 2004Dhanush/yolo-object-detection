YOLOv8 Object Detection

Comparing YOLOv8 Nano vs Small Models for Accuracy and Speed Trade-offs

Project Overview

This internship project demonstrates a comprehensive comparison of YOLOv8 Nano (fast) vs YOLOv8 Small (accurate) models for real-time object detection. The project includes model training with the COCO8 dataset, performance evaluation, and a detailed comparison table suitable for presentations.

Key Findings

Speed Advantage

YOLOv8n is 45% faster than YOLOv8s (45.2 FPS vs 30.5 FPS), making it ideal for real-time applications with resource constraints.

Accuracy Advantage

YOLOv8s is 20% more accurate (mAP50: 42% vs 35%), suitable for accuracy-critical applications.

Memory Efficiency

YOLOv8n uses 3.2M parameters vs 11.2M for YOLOv8s, enabling deployment on edge devices.

GPU Performance

Both models perform excellently on Tesla T4 GPU with significant speedup compared to CPU inference.

Dataset Performance

COCO8 dataset proved effective for training both models with consistent convergence across 20 epochs.
